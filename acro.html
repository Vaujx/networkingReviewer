OSI Model Related Acronyms

Acronym Full Form Description
OSI Open System Interconnection A conceptual model that standardizes the communication functions of a telecommunication or computing system
ISO International Organization for Standardization An international standard-setting body composed of representatives from various national standards organizations
LLC Logical Link Control The upper sublayer of the Data Link Layer that provides flow control and multiplexing for the logical link
MAC Media Access Control The lower sublayer of the Data Link Layer that controls how devices in a network gain access to data and permission to transmit it
PDU Protocol Data Unit Information that is delivered as a unit among peer entities of a network and that may contain control information, address information, or data

Protocol Acronyms

Acronym Full Form Description
HTTP Hypertext Transfer Protocol An application protocol for distributed, collaborative, hypermedia information systems that allows users to communicate data on the World Wide Web
FTP File Transfer Protocol A standard communication protocol used for the transfer of computer files from a server to a client on a computer network
SMTP Simple Mail Transfer Protocol An email protocol used for sending email messages from one email account to another via the internet
TELNET Teletype Network A network protocol used to virtually access a computer and to provide a two-way, collaborative channel between two machines
TFTP Trivial File Transfer Protocol A protocol that can be used to transfer files over networks and between computers using UDP
TCP Transmission Control Protocol A connection-oriented protocol that provides reliable, ordered, and error-checked delivery of data
UDP User Datagram Protocol A connectionless protocol that is used for time-sensitive applications like gaming, playing videos, or DNS lookups
IP Internet Protocol The principal communications protocol for relaying datagrams across network boundaries
DNS Domain Name System The Internet's system for converting alphabetic names into numeric IP addresses
DHCP Dynamic Host Configuration Protocol A protocol for automatically assigning IP addresses and other configurations to devices when they connect to a network
RPC Remote Procedure Call A protocol that provides the high-level communications paradigm used in the operating system
SQL Structured Query Language A language used to communicate with a database
NFS Network File System A networking protocol for distributed file sharing
RIP Routing Information Protocol A distance-vector routing protocol that uses hop count as a routing metric
IGRP Interior Gateway Routing Protocol A Cisco proprietary distance-vector routing protocol
OSPF Open Shortest Path First A routing protocol for IP networks that uses a link-state routing algorithm
EIGRP Enhanced Interior Gateway Routing Protocol An advanced distance-vector routing protocol that uses elements of link-state protocols
PPP Point-to-Point Protocol A data link protocol used to establish a direct connection between two nodes
HDLC High-Level Data Link Control A bit-oriented code-transparent synchronous data link layer protocol
IPX Internetwork Packet Exchange A networking protocol from Novell NetWare used in local area networks

Networking Acronyms

Acronym Full Form Description
LAN Local Area Network A network that connects a relatively small number of machines in a relatively close geographical area
WAN Wide Area Network A network that connects local-area networks over a potentially large geographic distance
MAN Metropolitan Area Network The communication infrastructures that have been developed in and around large cities
VPN Virtual Private Network A technology that creates a safe and encrypted connection over a less secure network
ISP Internet Service Provider A company that provides individuals and organizations access to the Internet
WWW World Wide Web An information system where documents and other resources are identified by URLs and accessible via the Internet
URL Uniform Resource Locator The address of a specific webpage or file on the Internet
NAT Network Address Translation A method of remapping one IP address space into another by modifying network address information

Hardware Acronyms

Acronym Full Form Description
NIC Network Interface Card A hardware component that connects a computer to a computer network
PCI Peripheral Component Interconnect A local computer bus for attaching hardware devices in a computer
USB Universal Serial Bus An industry standard that establishes specifications for cables, connectors, and protocols
Wi-Fi Wireless Fidelity A technology that allows electronic devices to connect to a wireless LAN network
ENIAC Electronic Numerical Integrator and Computer The first electronic general-purpose computer
VCR Video Cassette Recorder An electromechanical device for recording and playing back video on a cassette
DVD Digital Video Disc or Digital Versatile Disc A digital optical disc storage format
FAX Facsimile The telephonic transmission of scanned printed material

Data Formats and Encoding Acronyms

Acronym Full Form Description
ASCII American Standard Code for Information Interchange A character encoding standard for electronic communication
EBCDIC Extended Binary Coded Decimal Interchange Code An 8-bit character encoding used mainly on IBM mainframe and midrange computer operating systems
JPEG Joint Photographic Experts Group A commonly used method of lossy compression for digital images
MPEG Moving Picture Experts Group A working group that develops standards for audio and video compression and transmission
BMP Bitmap A raster graphics image file format used to store bitmap digital images
MIDI Musical Instrument Digital Interface A technical standard that describes a communications protocol, digital interface, and electrical connectors for music devices
WAV Waveform Audio File Format An audio file format standard for storing an audio bitstream on PCs
MP3 MPEG Audio Layer III A coding format for digital audio that uses a form of lossy data compression
CRC Cyclic Redundancy Check An error-detecting code used to detect accidental changes to raw data
